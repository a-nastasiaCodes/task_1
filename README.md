## Задача 1. Фильтрация чисел
**Функция `filter_numbers(numbers, threshold=0, greater=True)`**

- **Назначение:** фильтрует список чисел по заданному порогу.
- **Параметры:**
  - `numbers` (обязательный) – список чисел (целых или вещественных).
  - `threshold` (необязательный) – пороговое значение, по умолчанию `0`.
  - `greater` (необязательный) – булево значение. Если `True` (по умолчанию), оставляем числа **больше** порога; если `False` – числа **меньше** порога.
- **Возвращаемое значение:** новый список, содержащий только отфильтрованные числа. Если исходный список пуст, возвращается пустой список.
- **Пример:** 
  - `filter_numbers([1, -5, 10, 0, 3], threshold=0, greater=True)` вернёт `[1, 10, 3]`.
  - `filter_numbers([1, -5, 10, 0, 3], threshold=0, greater=False)` вернёт `[-5]`.

## Задача 2. Площадь прямоугольника
**Функция `rectangle_area(a, b=None)`**

- **Назначение:** вычисляет площадь прямоугольника или квадрата.
- **Параметры:**
  - `a` (обязательный) – длина первой стороны (число).
  - `b` (необязательный) – длина второй стороны. Если `b` не передан или равен `None`, то фигура считается квадратом со стороной `a`.
- **Возвращаемое значение:** площадь (число). Если `a` или `b` (когда задан) меньше или равны `0`, функция возвращает `0` (признак некорректных данных).
- **Примечание:** предполагается, что стороны могут быть как целыми, так и вещественными числами.
